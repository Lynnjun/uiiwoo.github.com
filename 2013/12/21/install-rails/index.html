MAC OSX 10.9.1 下安装 Rails

安装 RVM

RVM 是 RUBY 的管理工具，我们执行如下命令安装：

bash -s stable << (curl -s https://raw.github.com/wayneeseguin/rvm/master/binscripts/rvm-installer)

⚉ 安装完成后需要在 ~/.bash_profile 中加入下面信息：

[[ -s "/Users/bjsky/.rvm/scripts/rvm" ]] && source "/Users/bjsky/.rvm/scripts/rvm"

很简单，到此就算完成了 RVM 的安装。

解决问题

遇到的麻烦多的差点让我想放弃，这里仅仅记录了我遇到的比较重点的几个问题和我的解决办法，开篇之前重点说明下，血泪教训：

其实我所遇到的最大的问题是我在装载的第二个磁盘宗卷上用的 extFat 文件格式，之后将个人路径默认设置给了该宗卷，结果导致了一大堆各种各样的权限问题和莫名其妙的错误，如果看家也有这样的问题，建议不用急着排插当前的错误了，先将宗卷文件格式改回去在说吧，Linux 下同理。

GCC 无法安装的问题

⚉ 需要 CLT：

xCode 5.0 后 command line tools 是使用命令安装的，也可以直接到苹果开发者网站下载安装包安装。

安装包下载地址：https://developer.apple.com/downloads/index.action
安装命令：xcode-select --install

⚉ 需要需要 GCC：

直接用 brew 安装 apple-gcc42 ，如果想要安装 gcc49 或更新，直接去下载安装吧，brew 现在装不了了。

⚉ 需要解决 Homebrew 搜索软件的警告问题：

警告提示，需要 Git 的 SSH KEY ，所以用 ssh 命令生成 key ，并添加到自己的 Github 账户设置里面。

首先在本地创建ssh key：
      ssh-keygen -t rsa -C "your_email@youremail.com"
  后面的 your_email@youremail.com 改为你的邮箱，之后会要求确认路径和输入密码，可用使用默认的一路回车就行。

成功的话会在 ~/ 下生成.ssh文件夹，进去，打开id_rsa.pub，复制里面的key。提示，可用 cat 命令，也可用 vim 打开。

回到github，进入 Account Settings，左边选择 SSH Keys -> Add SSH Keyz。title 随便填，粘贴 key。

验证是否成功，可用在 git bash 下输入：
      ssh -T git@github.com

如果是第一次连接，会提示是否 continue，输入 yes 就会看到：You’ve successfully authenticated, but GitHub does not provide shell access。这就表示已成功连上github。

⚉ 需要解决与 github 连接不成功，提示权限出错的问题在在：

警告信息：UNPROTECTED PRIVATE KEY FILE!
解决办法，修改权限：
      chmod 755 ~/.ssh/
      chmod 600 ~/.ssh/id_rsa ~/.ssh/id_rsa.pub 
      chmod 644 ~/.ssh/known_hosts

题外：我在这里遇到了权限无法修改的问题，因为我 MAC 下的默认路径是在挂载的硬盘分区上的，该分区是 extFat 格式，不能被修改权限。。。我嘞个去啊，我的个人默认配置路径和众多数据都在这盘上面了，由于一年多以前第一次玩 MAC 系统，挂载的硬盘设置成 extFat 方便我直接到 win下考数据的，一直也没觉得有什么太严重的问题，无非就是开关机慢点、索引文件慢点、Aperture 的图库不能放上面等问题，由于数据多，个人资料路径都设定了，就不想改了，结果在这儿等着我还债。。。

⚉ 需要解决执行 rvm requirements 的各种报错：

我安装了 apple-gcc42 后还提示错误，查看发现是 Homebrew 我设置的路径下面有中文，rvm 识别乱码。将 Homebrew 修改为默认推荐的 usr/local 后解决问题。

安装 Ruby

⚉ 需要查看已知的 Ruby 版本列表：

命令：rvm list known

⚉ 需要安装最新的 Ruby 版本：

命令：rvm  install  ruby  --head

⚉ 需要安装指定版本：

命令：rvm  install  2.0.0

⚉ 需要指定当前使用的 Ruby 版本：

命令：rvm  use  2.0.0  --default

安装 Rails

⚉ 需要使用 gem 安装 Rails：

命令：gem install rails -V

End
