利用 Homebrew 管理 Mac 平台下的软件包

什么是 Homebrew

Homebrew 是 Mac OS X 平台下的软件包管理系统。

Mac OS X 是基于 Unix 的操作系统，可以安装大部分为 Unix/Linux 开发的软件。然而，如果只是以使用为目的，对每个软件都进行手工编译不是很方便，也不利于管理已安装的软件，于是出现了类似于 Linux 中 APT、Yum 等类似的软件包管理系统，其中最著名的有 MacPorts、Fink、Homebrew 等。

Homebrew 的安装

首先确保你的系统满足如下要求：

  • x86 的 U
  • 操作系统为 Mac OS X 雪豹以上
  • 安装 Git（Mac OS X 狮子已预装 Git）
  • 安装 Xcode 开发工具
    注：Xcode 4.3中，命令行编译工具是可选安装，需要在Preferences > Downloads中激活。
  • 安装 Java Developer Update
    注：可选，Homebrew本身不依赖于Java，只有部分软件包的安装需要Java支持。

下载 HomeBrew

  • 地址： https://nodeload.github.com/mxcl/homebrew/legacy.tar.gz/master
  • 下载最方便也可重复使用，下载好后直接解压缩，可以放在任何你想放的地方，推荐放到 /usr/local 文件夹里面。
  • 注意一下，任意放置的话，路径不要有中文字符，不然 rvm 安装 ruby 更新时会出错。

设置环境变量

需要将 homebrew/bin/ 文件夹写入环境变量，在这里我是直接加入到全局的 ~/.bash_profile 的 PATH 里面。

⚉ .bash_profile 的内容：

  export PATH=/opt/local/bin:/opt/local/sbin:/opt/local/bin:/opt/local/sbin:/usr/bin:/bin:/usr/sbin:/sbin:/usr/local/bin:/usr/local/mysql/bin:/Applications/adt-bundle-mac-x86_64/android-ndk-r8d:/Applications/adt-bundle-mac-x86_64/sdk:/Applications/adt-bundle-mac-x86_64/cocos2d-2.1beta3-x-2.1.0:/usr/local/homebrew/bin:$PATH

Mac 下如何设置 PATH

⚉ 查看 PATH：

  命令：$PATH

⚉ 设置 PATH：

  命令：echo "export PATH=xxxxxx:$PATH" >> ~/.bash_profile
  解释：把"export PATH=xxxxxx:$PATH"输出打印到~/.bash_profile中去。

⚉ ~/.bash_profile 介绍：

  mac 和 linux 终端一般用 bash 来进行解析。当bash在读完了整体环境变量的 /etc/profile 并借此调用其它配置文件后，接下来则是会读取用户自定义的配置文件。bash 读取的文件总共有三种：
  　     ~/.bash_profile 　　~/.bash_login  　　~/.profile
  其实 bash 再启动是只读上面文件的一个，而读取的顺序则是依照上面的顺序。也就是说读到 bash_profile 就不读后面的了，如果 bash_profile 不存在，后面的才能有机会。

⚉ 查看 bash_profile 里面有什么内容：

  命令：cat ~/.bash_profile

重新打开终端，已经可以使用 brew

使用 Homebrew

⚉ 建立本地的 git 仓库，执行如下：

  cd /usr/local
  git init
  git remote add origin git://github.com/mxcl/homebrew.git
  git pull origin master

⚉ 查找和安装软件：

  brew search #软件名#
  brew install #软件名#

⚉ GitHub 上的项目,可直接克隆下来：

  git clone http://github.com/#你的github用户名#/homebrew.git/tmp/homebrew

⚉ Homebrew 其它常用指令：

  brew list   ## 列出已安装的软件
  brew update   ## 更新Homebrew
  brew home *  ## 用浏览器打开
  brew info *  ## 显示软件内容信息
  brew deps *  ## 显示包依赖
  brew server *   ## 可以通过浏览器访问 http://localhost:4567 来同网页来管理包
  brew -h brew   ## 帮助

删除Homebrew

⚉ 卸载指令：

  cd `brew –prefix`
  rm -rf Cellar
  brew prune
  rm -rf Library .git .gitignore bin/brew README.md share/man/man1/brew
  rm -rf ~/Library/Caches/Homebrew

End
